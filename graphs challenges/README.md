# Graph Algorithms

This folder contains implementations of advanced network and graph theory algorithms studied during the freeCodeCamp curriculum.

## Concepts & Algorithms Covered
* **Adjacency Lists vs Matrices:** Converting structural representations of data graphs.
* **Dijkstra's Shortest Path Algorithm:** An $O(V^2)$ pathfinding algorithm using a greedy approach to find the shortest routing distance between nodes.
* **Breadth-First Search (BFS):** A fundamental layer-by-layer graph traversal algorithm utilizing a queue data structure.
* **Depth-First Search (DFS):** A deep-dive graph traversal algorithm utilizing a stack data structure to explore paths to their absolute end before backtracking.
