# Sorting Algorithms

This folder contains implementations of core sorting algorithms studied during the freeCodeCamp curriculum.

## Algorithms Implemented
* **Bubble Sort:** $O(n^2)$ time complexity. Swaps adjacent elements.
* **Selection Sort:** $O(n^2)$ time complexity. Tracks minimum values.
* **Insertion Sort:** $O(n^2)$ time complexity. Shifts elements backward.
* **Quicksort:** $O(n \log n)$ average time complexity. Uses pivot partitioning.
